Concept


I created a weather app using pure JS for the SheCodes Plus course. The goal of the project was to create a weather application with features such as an upcoming forecast, a search engine for cities, and a geolocation option for local weather. I used Open Weather API for weather and forecast information and employed Axios to handle AJAX requests.

Reflection


Learning Experience


Learning the development process
I learned how to create a weather application from start to finish: wireframes, building the HTML and CSS wireframes, using APIs, implementing functionalities with JavaScript, and hosting from Netlify.

APIs and Axios.
I learned how to use Axios to make API requests from Open Weather API in order to get data for weather and forecast information.
