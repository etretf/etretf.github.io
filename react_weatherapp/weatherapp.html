Concept


I created a weather app for the SheCodes React course. The goal of the project was to create a weather application with features such as an upcoming forecast up to five days and having a search engine for cities. I used Open Weather API for weather and forecast information and employed Axios to handle AJAX requests.

Reflection


Learning Experience


Learning React.
I learned how to use React to build my user interface with reusable UI components and learned how to build a React App.

APIs and Axios.
I learned how to use Axios to make API requests from Open Weather API in order to get data for weather and forecast information.

Next Steps

I only developed the app to work on desktop interfaces. The next step would be to make it responsive and display properly on mobile screens.

I also wanted to add a location feature to get the weather and forecast of the user's location, but I did not add that feature in the scope of the project.
